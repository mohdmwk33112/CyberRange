3. Authentication & Authorization Service
Purpose

Handles user registration, login, and authorization.

Database

MongoDB (Users Collection)

Responsibilities

Register users

Login users

Issue JWT tokens

Role-based access control (student, admin, instructor)

Implementation Steps

Create Auth microservice (NestJS)

Define User schema:

Username

Email

Password (hashed)

Role

Implement endpoints:

POST /register

POST /login

Use:

bcrypt for password hashing

JWT for authentication

Return signed token to API Gateway
